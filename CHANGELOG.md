# 변경 이력

밥크로스(Bab-Cross)의 사용자에게 의미 있는 기능 변화와 개선 사항을 기록합니다.

## v0.1.105 - 2026-06-11

### 주요 변경사항

- 밥판 초대 코드를 틀리게 입력하면 흔들림과 진동 피드백 후 4자리 입력값을 자동으로 비우도록 개선했습니다.
- 사용자가 백스페이스로 지우지 않고 바로 코드를 다시 입력할 수 있게 했습니다.
- 앱 버전을 `0.1.105`로 올렸습니다.

## v0.1.104 - 2026-06-11

### 주요 변경사항

- 밥판 초대 코드 오류 시 4자리 입력칸 흔들림을 더 크고 선명하게 조정했습니다.
- 잘못된 코드 입력 피드백에 두 번 짧게 울리는 진동 패턴을 추가했습니다.
- 앱 버전을 `0.1.104`로 올렸습니다.

## v0.1.103 - 2026-06-11

### 주요 변경사항

- 밥판 초대 코드가 틀리면 4자리 입력칸이 짧게 흔들리도록 피드백을 추가했습니다.
- 잘못 입력한 순간을 더 쉽게 알아차릴 수 있도록 가벼운 햅틱 피드백도 함께 적용했습니다.
- 앱 버전을 `0.1.103`으로 올렸습니다.

## v0.1.102 - 2026-06-11

### 주요 변경사항

- 밥판 초대 화면에서 남은 시간 카드를 코드 입력 카드 위로 배치했습니다.
- 초대 안내 문구의 남은 시간 텍스트를 제거해 시간 표시 중복을 줄였습니다.
- 앱 버전을 `0.1.102`로 올렸습니다.

## v0.1.101 - 2026-06-11

### 주요 변경사항

- 밥판 초대 코드 입력 중 숫자 키보드가 올라와도 4자리 입력칸이 보이도록 화면 리프트 처리를 보강했습니다.
- 초대 코드 입력칸은 필요한 만큼만 위로 올리고, 하단 메뉴 후보 입력은 기존처럼 화면 아래 보정을 유지하도록 키보드 스크롤 처리를 분리했습니다.
- 앱 버전을 `0.1.101`로 올렸습니다.

## v0.1.100 - 2026-06-11

### 주요 변경사항

- 밥판 초대 코드 입력을 4자리 슬롯 형태로 표시해 어떤 자리까지 입력했는지 더 분명하게 보이도록 개선했습니다.
- 4칸 영역을 탭하면 숫자 키보드가 열리고, 코드가 맞으면 기존처럼 바로 밥판에 참여합니다.
- 앱 버전을 `0.1.100`으로 올렸습니다.

## v0.1.99 - 2026-06-11

### 주요 변경사항

- 결정 카드 등급 종류를 처음 설계 기준인 `초기 순차 공개 + 이후 가중 확률` 방식으로 되돌렸습니다.
- 등급명은 `보통`, `만족`, `희귀`, `전설` 명칭을 유지하고, 별점은 계속 응답 순서 기준으로 분리했습니다.
- 앱 버전을 `0.1.99`로 올렸습니다.

## v0.1.98 - 2026-06-11

### 주요 변경사항

- 밥판을 열 때 4자리 초대 코드를 생성하고 밥판장 화면에 크게 표시하도록 했습니다.
- 초대받은 사용자는 4자리 코드를 맞게 입력하면 별도 참여 버튼 없이 바로 밥판에 참여하고 메뉴 선택 화면으로 이동합니다.
- 코드 입력 전에는 초대 상세와 참여 흐름을 잠가 야외에서 불특정 사용자가 밥판에 들어오는 일을 줄였습니다.
- 앱 버전을 `0.1.98`로 올렸습니다.

## v0.1.97 - 2026-06-11

- 홈 화면의 최근 결정 카드와 상단 영역을 정리해 중복 문구와 불필요한 스크롤 동작을 줄였습니다.
- 아바타 표시를 원형 테두리 안에 꽉 차게 맞추고, 아바타 선택 시트의 배경 잘림이 덜 보이도록 개선했습니다.

- 밥판 열기의 제한시간 선택 그룹을 하나의 연결된 버튼처럼 보이도록 다듬었습니다.
