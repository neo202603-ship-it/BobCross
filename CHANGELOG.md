# 변경 이력

밥크로스(Bab-Cross)의 사용자에게 의미 있는 기능 변화와 개선 사항을 기록합니다.

## v0.1.2 - 2026-05-30

### 주요 변경사항

- 메뉴 후보, 투표 판세, 결과 카드에 대표 1인분 기준의 대략 칼로리 힌트를 추가했습니다.
- 메뉴 결정 팝업을 밥크로스가 말하는 이미지 컷 형태로 바꾸고, 10초 후 자동으로 사라지게 했습니다.
- 결정 컷신은 사용자가 탭하거나 바깥 영역을 누르면 즉시 닫히도록 했습니다.
- 홈의 진행 중인 밥판 카드를 최근 결정 카드 스타일로 맞추고, 남은 시간/밥친구/연결 중복 문구를 제거했습니다.
- 내가 연 밥판 상단 카드에서 후보 수, 선택 인원, 남은 시간 중복 정보를 제거했습니다.
- 앱 버전을 `0.1.2`로 올렸습니다.

## v0.1.1 - 2026-05-30

### 주요 변경사항

- 결과 화면의 결정 카드를 공유 이미지에 어울리도록 고정 크기 카드 중심으로 재구성했습니다.
- 결정 메뉴를 더 크게 보여주고, 질문은 카드 상단 영역으로 옮겨 결과 집중도를 높였습니다.
- 카드 등급을 일반/희귀/레어/전설 느낌의 게임 아이템 표현으로 다듬고, 초기 4회 이후에는 등급별 노출 빈도가 다르게 적용되도록 했습니다.
- 카드 테두리에 시계방향 회전 하이라이트를 적용하고, 양쪽 빛줄기가 천천히 움직이도록 조정했습니다.
- 희귀/전설 카드에는 우측 하단 둥근 정사각형 홀로그램 스티커를 추가했습니다.
- 결정 카드 안의 바코드를 한 줄 구성으로 정리하고 높이를 키워 카드 하단에 자연스럽게 녹였습니다.
- 결과 카드 좌측에 밥반장 아바타가 일부 보이는 포인트 영역을 추가해 카드별 식별감을 높였습니다.
- 좌우 스와이프로 지난 결정 카드를 넘겨 볼 수 있도록 결과 카드 탐색을 개선하고, 세로 스크롤과 충돌하지 않게 조정했습니다.
- 메뉴 선택 없이 끝난 밥판은 `결정 보류` 상태로 보여주고, 지도 찾기 등 선택 메뉴 전용 동작은 숨기도록 했습니다.
- 홈 화면의 최근 결정 카드와 상단 영역을 정리해 중복 문구와 불필요한 스크롤 동작을 줄였습니다.
- 아바타 표시를 원형 테두리 안에 꽉 차게 맞추고, 아바타 선택 시트의 배경 잘림이 덜 보이도록 개선했습니다.
- 내 밥닉 화면 문구를 `내 밥닉`으로 정리하고, 밥닉 제안 시 제안된 사물 이름에 맞는 아바타가 자동 선택되도록 했습니다.
- 밥판 열기의 제한시간 선택 그룹을 하나의 연결된 버튼처럼 보이도록 다듬었습니다.
- 앱 버전을 `0.1.1`로 올렸습니다.

## v0.1.0 - 2026-05-26

### 주요 변경사항

- `near-vote-android`의 Nearby Connections 기반 근거리 선택 엔진을 밥크로스 앱으로 재구성했습니다.
- 앱 이름, applicationId, namespace, APK 이름을 `Bab-Cross` 기준으로 변경했습니다.
- Nearby serviceId와 SharedPreferences 이름을 분리해 기존 참조 앱 설치본과 섞이지 않도록 했습니다.
- 홈, 밥판 열기, 주변 밥신호, 메뉴 선택, 결과 화면의 주요 문구를 밥크로스 세계관으로 교체했습니다.
- 아프로 파마머리 밥공기 마스코트 이미지를 생성해 런처 아이콘과 홈 화면에 적용했습니다.
- 메뉴가 결정되면 `어때요, 메뉴 고르기 참 쉽죠?` 팝업이 표시되도록 했습니다.
- 기본 템플릿을 점심/회식/후식 중심의 밥판 템플릿으로 바꿨습니다.
- 주 행동 색상을 밥색, 고추장 레드, 따뜻한 식탁 계열로 조정했습니다.
- README를 밥크로스 기준으로 다시 작성했습니다.
