# 변경 이력

밥크로스(Bab-Cross)의 사용자에게 의미 있는 기능 변화와 개선 사항을 기록합니다.

## v0.1.29 - 2026-05-30

### 주요 변경사항

- 밥판 열기 화면 상단에 `혼자 먼저 정리`와 `같이 고르기` 모드를 선택하는 세그먼트를 추가했습니다.
- 선택한 모드에 따라 안내 문구와 하단 주 버튼 문구가 바뀌도록 했습니다.
- `혼자 먼저 정리` 모드에서는 연결된 밥친구가 없어도 접속자 없음 확인 팝업 없이 바로 밥판을 열 수 있게 했습니다.
- 앱 버전을 `0.1.29`로 올렸습니다.

## v0.1.28 - 2026-05-30

### 주요 변경사항

- 오늘의 밥결정 공유 버튼을 누르면 실제 공유 이미지와 문구를 먼저 확인하는 미리보기 화면이 열리도록 했습니다.
- 공유 미리보기에서 `이미지 공유`와 `문구만 복사`를 선택할 수 있게 했습니다.
- 공유 이미지의 카드 안전 영역을 상수화하고, 긴 질문/메뉴/후보/이유 문구는 자동으로 글자 크기를 줄인 뒤 필요한 경우 말줄임 처리하도록 보강했습니다.
- 앱 버전을 `0.1.28`로 올렸습니다.

## v0.1.27 - 2026-05-30

### 주요 변경사항

- 설정 화면에 `밥판 기본값` 섹션을 추가해 기본 제한시간, 밥친구 후보 추가, 밥친구별 선택 공개 기본값을 저장할 수 있게 했습니다.
- 새 밥판을 열 때 저장한 기본값이 자동으로 적용되도록 했습니다.
- `개발자 진단` 화면을 `밥친구 찾기 점검`으로 바꾸고, 주요 액션을 밥친구 다시 찾기, 권한 다시 확인, 설정 열기, 밥판 체험하기 중심으로 정리했습니다.
- 연결 상세 로그는 기본적으로 숨기고 필요할 때만 펼쳐 보도록 조정했습니다.
- 앱 버전을 `0.1.27`로 올렸습니다.

## v0.1.26 - 2026-05-30

### 주요 변경사항

- 오늘의 밥결정 공유 이미지에서 후보와 결정 이유 텍스트가 카드 테두리 밖으로 밀리지 않도록 하단 정보 영역을 위로 올리고 본문 크기를 조정했습니다.
- 앱 버전을 `0.1.26`으로 올렸습니다.

## v0.1.25 - 2026-05-30

### 주요 변경사항

- 밥판장이 혼자 밥판을 열어도 한 표를 고른 뒤 `오늘의 밥결정 만들기`로 바로 결과를 만들 수 있게 안내 문구를 정리했습니다.
- 연결된 밥친구가 없을 때의 확인 팝업을 `혼자 먼저 밥판 열기` 흐름으로 바꿔 후보 정리와 결과 공유 가치를 더 명확히 했습니다.
- 오늘의 밥결정 공유 이미지와 공유 문구에 메뉴 후보, 득표 요약, 결정 이유를 함께 담도록 강화했습니다.
- 앱 버전을 `0.1.25`로 올렸습니다.

## v0.1.24 - 2026-05-30

### 주요 변경사항

- 내 밥닉 화면의 `밥닉 제안` 버튼을 `현재 밥닉` 라벨 오른쪽 작은 보조 액션으로 옮겨 입력 흐름을 더 자연스럽게 정리했습니다.
- 앱 버전을 `0.1.24`로 올렸습니다.

## v0.1.23 - 2026-05-30

### 주요 변경사항

- 설정 화면 상단에 아바타, 밥닉, 현재 연결 요약을 보여주는 프로필 카드를 추가했습니다.
- Bluetooth, Wi-Fi, 위치/Nearby 권한 상태를 한 번에 확인하고 권한 허용, 설정 열기, 밥판 체험하기로 이어질 수 있는 연결 준비 상태 카드를 추가했습니다.
- 자동 연결을 큰 실행 버튼 대신 토글형 설정 카드로 바꿔 설정 화면의 행동 우선순위를 정리했습니다.
- 앱 버전을 `0.1.23`으로 올렸습니다.

## v0.1.22 - 2026-05-30

### 주요 변경사항

- 밥판 열기 화면에서 하단 고정 `밥신호 보내기` 버튼이 마지막 보조 버튼을 가리지 않도록 스크롤 하단 여백 계산을 수정했습니다.
- `밥판 저장` 버튼명을 `템플릿 저장`으로 바꿔 실제 밥판 전송과 재사용용 저장의 차이를 더 명확히 했습니다.
- 앱 버전을 `0.1.22`로 올렸습니다.

## v0.1.21 - 2026-05-30

### 주요 변경사항

- 3차 개선안을 문서화하고, 밥판장이 다음 행동을 더 쉽게 이해하도록 `내가 연 밥판` 상태 흐름 카드를 추가했습니다.
- 밥판 생성 화면의 `밥신호 보내기`를 하단 고정 버튼으로 옮겨 입력 중에도 주요 행동이 보이도록 개선했습니다.
- 공동 1등 결과 카드에서 `공동 1등`을 먼저 보여주고 동률 메뉴 목록을 보조 줄로 분리했습니다.
- 체험 모드를 `밥판 생성 → 가상 밥친구 선택 → 결과 카드 공개` 3단계 흐름으로 정리했습니다.
- 앱 버전을 `0.1.21`로 올렸습니다.

## v0.1.20 - 2026-05-30

### 주요 변경사항

- 결과 화면에서 공동 1등 메뉴가 2개 이상일 때 밥크로스 추천 멘트를 표시하도록 추가했습니다.
- 동률 상황에서는 밥판장 선택, 내 선택, 후보 순서 기준으로 추천 메뉴를 골라 `밥크로스는 OOO(으)로 결정하면 어떨까 해요.`라고 안내합니다.
- 앱 버전을 `0.1.20`으로 올렸습니다.

- 공개 결과에서는 각 밥친구의 아바타, 밥닉, 선택한 메뉴가 함께 보이도록 정리했습니다.

## v0.1.17 - 2026-05-30

- 밥크로스 이미지를 왼쪽 배경처럼 은은하게 배치하고, `밥판 열기` 버튼을 사용자 아바타 아래로 옮겼습니다.

## v0.1.15 - 2026-05-30

- 홈의 밥판 체험 카드를 강화하고, 가상 밥친구가 참여한 생성/선택/결과 카드 흐름을 보여주도록 개선했습니다.

- 오늘의 밥결정 카드를 이미지로 만들어 공유할 수 있는 액션을 추가했습니다.
- 앱 버전을 `0.1.15`로 올렸습니다.

## v0.1.2 - 2026-05-30

- 내가 연 밥판 상단 카드에서 후보 수, 선택 인원, 남은 시간 중복 정보를 제거했습니다.

- 결과 화면의 결정 카드를 공유 이미지에 어울리도록 고정 크기 카드 중심으로 재구성했습니다.
- 결정 메뉴를 더 크게 보여주고, 질문은 카드 상단 영역으로 옮겨 결과 집중도를 높였습니다.
- 카드 등급을 일반/희귀/레어/전설 느낌의 게임 아이템 표현으로 다듬고, 초기 4회 이후에는 등급별 노출 빈도가 다르게 적용되도록 했습니다.
- 카드 테두리에 시계방향 회전 하이라이트를 적용하고, 양쪽 빛줄기가 천천히 움직이도록 조정했습니다.
- 희귀/전설 카드에는 우측 하단 둥근 정사각형 홀로그램 스티커를 추가했습니다.
- 결정 카드 안의 바코드를 한 줄 구성으로 정리하고 높이를 키워 카드 하단에 자연스럽게 녹였습니다.
- 결과 카드 좌측에 밥반장 아바타가 일부 보이는 포인트 영역을 추가해 카드별 식별감을 높였습니다.
- 좌우 스와이프로 지난 결정 카드를 넘겨 볼 수 있도록 결과 카드 탐색을 개선하고, 세로 스크롤과 충돌하지 않게 조정했습니다.
- 메뉴 선택 없이 끝난 밥판은 `결정 보류` 상태로 보여주고, 지도 찾기 등 선택 메뉴 전용 동작은 숨기도록 했습니다.
- 홈 화면의 최근 결정 카드와 상단 영역을 정리해 중복 문구와 불필요한 스크롤 동작을 줄였습니다.
- 아바타 표시를 원형 테두리 안에 꽉 차게 맞추고, 아바타 선택 시트의 배경 잘림이 덜 보이도록 개선했습니다.
- 내 밥닉 화면 문구를 `내 밥닉`으로 정리하고, 밥닉 제안 시 제안된 사물 이름에 맞는 아바타가 자동 선택되도록 했습니다.
- 밥판 열기의 제한시간 선택 그룹을 하나의 연결된 버튼처럼 보이도록 다듬었습니다.
- 앱 버전을 `0.1.1`로 올렸습니다.
