# 밥크로스(Bab-Cross) 개인정보처리방침

시행일: 2026-06-02

## 한 줄 요약

밥크로스는 서버가 없습니다. 개발자 서버로 개인정보를 수집하지 않으며, 밥판 기능에 필요한 정보는 내 기기 안에 저장되거나 사용자가 참여한 가까운 밥판 기기끼리만 전달됩니다.

## 1. 개발자 서버로 수집하는 개인정보

없습니다.

밥크로스는 회원가입, 전화번호, 이메일, 위치 기록, 연락처, 광고 ID, 사용 분석 데이터를 개발자 서버나 광고/분석 SDK로 수집하지 않습니다.

## 2. 기기 안에 저장되는 정보

다음 정보가 앱 사용을 위해 내 기기 안에 저장될 수 있습니다.

- 밥닉네임
- 아바타
- 밥판 템플릿
- 선택 영수증
- 지난 결과

이 정보는 앱 기능을 위한 로컬 저장용이며 밥크로스 서버로 올라가지 않습니다.

## 3. 주변 기기와 공유되는 정보

사용자가 밥판을 열거나 참여하면 다음 정보가 Nearby Connections로 가까운 참여 기기에 전달될 수 있습니다.

- 밥닉네임
- 아바타
- 메뉴 후보
- 선택 내용
- 결과

이는 앱 기능 수행을 위한 사용자 주도 근거리 공유이며, 밥크로스 서버 수집이나 공개 인터넷 전송이 아닙니다.

## 4. 권한 사용

주변 기기 연결을 위해 Android가 다음 계열의 권한을 요청할 수 있습니다.

- Bluetooth
- Wi-Fi / Nearby 기기
- 위치 관련 권한

이 권한은 Nearby Connections로 근처 밥친구를 찾고 연결하기 위한 것이며, 위치 기록을 수집하거나 저장하지 않습니다.

## 5. 제3자 제공과 광고

밥크로스는 제3자 제공, 광고 추적, 분석 SDK를 사용하지 않습니다.

## 6. 삭제

앱 데이터를 삭제하거나 앱을 제거하면 기기 안에 저장된 밥크로스 데이터도 삭제됩니다.

단, 이미 주변 기기에 공유된 결과는 해당 기기에서 따로 삭제해야 합니다.
