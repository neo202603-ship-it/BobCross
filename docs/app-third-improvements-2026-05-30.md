# 밥크로스 3차 개선안

작성일: 2026-05-30

## 목표

밥크로스의 다음 개선은 기능 추가보다 사용자가 지금 무엇을 해야 하는지 더 빨리 이해하게 만드는 데 둔다. 홈과 결과 카드의 개성은 유지하되, 밥판 생성, 진행, 결과 확인까지의 흐름을 더 또렷하게 정리한다.

## 작업 순서

1. `내가 연 밥판` 상태 흐름 정리
   - 선택 전에는 내 선택을 먼저 유도한다.
   - 선택 후에는 밥친구 선택 대기 또는 메뉴 결정 가능 상태를 보여준다.
   - 결과 공유 후에는 결정 메뉴 보기로 이어지게 한다.

2. 밥판 생성 화면 하단 sticky `밥신호 보내기`
   - 입력 도중에도 핵심 행동이 항상 보이게 한다.
   - 본문에는 `밥판 저장`만 남겨 보조 행동과 주요 행동을 분리한다.

3. 동률 결과 카드 표현 개선
   - 공동 1등일 때 중앙 카드에는 `공동 1등`을 먼저 보여준다.
   - 동률 메뉴 목록은 보조 줄로 보여준다.
   - 밥크로스 추천 멘트는 별도 카드로 유지해 확정과 추천이 섞이지 않게 한다.

4. 체험 모드 3단계 시뮬레이션화
   - 가상 밥판 생성, 가상 밥친구 선택, 결과 카드 공개 흐름을 한 화면에서 명확히 보여준다.
   - 체험 결과에서도 밥친구별 선택 목록을 보여 실제 흐름을 미리 이해하게 한다.

## 기대 효과

- 밥판장이 화면에서 다음 행동을 덜 고민한다.
- 결과 화면이 동률, 추천, 최종 결과를 더 명확히 구분한다.
- 첫 사용자도 권한 없이 체험 모드에서 앱의 핵심 재미를 빠르게 이해한다.
