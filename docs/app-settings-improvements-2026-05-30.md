# 밥크로스 설정 화면 개선안

작성일: 2026-05-30

## 현재 상태

현재 설정 화면은 `자동 연결`, `내 밥닉 관리`, `개인정보처리방침`, 앱 버전만 제공한다. 기능은 작동하지만, 사용자가 설정 화면에서 기대하는 `내 상태 확인`, `연결 문제 해결`, `기본값 관리`, `데이터 관리`가 분리되어 있지 않아 화면이 다소 비어 보인다.

## 개선 목표

설정 화면을 단순 옵션 목록이 아니라, 사용자가 밥크로스 사용 상태를 확인하고 문제를 해결하며 기본 사용 방식을 조정하는 관리 화면으로 바꾼다.

## 개선 우선순위

1. 프로필 요약 카드 추가
   - 아바타, 밥닉, 현재 연결 상태를 상단에 보여준다.
   - `내 밥닉 관리`는 독립 버튼보다 카드 안의 `수정` 액션으로 자연스럽게 진입하게 한다.
   - 설정 화면 첫인상을 `누구의 설정인지`가 보이는 화면으로 만든다.

2. 연결 준비 상태 카드 개선
   - Bluetooth, Wi-Fi, 위치/Nearby 권한 상태를 사용자 언어로 표시한다.
   - 문제가 있으면 `권한 허용`, `설정 열기`, `밥판 체험하기`를 바로 제공한다.
   - 자동 연결 상태와 실제 연결 준비 상태를 한 영역에서 이해할 수 있게 한다.

3. 자동 연결을 토글형 설정으로 변경
   - 현재 큰 primary 버튼인 `자동 연결 끄기/켜기`를 토글 카드 형태로 바꾼다.
   - 설명 문구는 `앱 실행 시 근처 밥친구를 자동으로 찾기`처럼 행동 중심으로 정리한다.
   - 위험한 주 액션처럼 보이지 않게 한다.

4. 밥판 기본값 섹션 추가
   - 기본 제한시간
   - 밥친구 메뉴 추가 허용 기본값
   - 결과에서 밥친구별 선택 공개 기본값
   - 자주 쓰는 템플릿 또는 최근 템플릿 진입
   - 매번 밥판 열기 화면에서 반복 조작하는 설정을 줄인다.

5. 데이터와 도움말 섹션 추가
   - 저장한 템플릿 관리
   - 결과 기록 보기
   - 온보딩 다시 보기
   - 밥판 체험하기
   - 기록 초기화는 필요하면 별도 확인 팝업을 붙여 후순위로 둔다.

6. 고급 진단 진입 정리
   - `개발자 진단`이라는 표현은 일반 사용자에게 노출하지 않는다.
   - 설정 메인에서는 `연결 문제 해결`로 진입하게 하고, 내부에 광고/탐색/로그 기능을 둔다.
   - 개발용 느낌을 줄이고 실제 문제 해결 화면처럼 보이게 한다.

## 제안 작업 순서

1. 설정 상단 프로필 요약 카드 구현
2. 연결 준비 상태 카드와 자동 연결 토글 카드 구현
3. 도움말/체험/온보딩 다시 보기 섹션 추가
4. 밥판 기본값 저장 구조 검토 및 구현
5. 고급 진단 화면 이름과 진입 방식 정리
